Créer un site web **multipage** en React avec Tailwind CSS, responsive (mobile, tablette, desktop), inspiré du design "Elderly Care 02" (https://websitedemos.net/elderly-care-02/?customizer=true&page-builder=elementor), pour le projet **LUMIÈRE — Hubs Modulaire Santé & Éducation (Microgrid-powered)**. Le site doit inclure les pages suivantes :

1️⃣ **Accueil**
- Hero section : image / illustration d’un hub LUMIÈRE + tagline « Un million pour un hub, 30 000 vies touchées par an ».
- Elevator pitch résumé 20 secondes
- Sections projet : santé, éducation, énergie
- Chiffres clés animés
- CTA « Contribuer maintenant » vers page Don

2️⃣ **À propos**
- Mission, vision, valeurs, équipe
- Gouvernance et sécurité des fonds
- Témoignages / médias

3️⃣ **Détails** (page principale avec sous-menu)
- Sous-menu : **Présentation** et **Budget**
- **Présentation** : description complète des Hubs LUMIÈRE, objectifs concrets, impacts tangibles
- **Budget** : répartition détaillée des 1 000 000 $ par hub, timeline, KPI, audit, dashboard d’impact

4️⃣ **Don / Devenir mécène**
- Formulaire sécurisé (Stripe, PayPal, virement)
- Progress bar collecté vs objectif
- Options contribution : hub entier, partie d’un hub, montant libre
- Valeur pour le mécène : naming, plaque, rapports trimestriels, visite VIP, co-branding

5️⃣ **Contact**
- Formulaire simple + coordonnées (mail, téléphone)

---

### Fonctionnalités générales
- Navbar et footer réutilisables, avec **sous-menu pour Détails** (Présentation, Budget)
- Animations légères pour chiffres clés et CTA
- Intégration médias : images, illustrations 3D / flat design
- Téléchargement PDF : Executive Summary + Pitch Deck
- Responsive design complet
- Préparation pour hébergement Vercel / Netlify

### Design & UX
- Palette couleurs : Bleu & Or
- Typographie : claire, élégante, professionnelle
- CTA visibles et répétés
- Sections modulaires lisibles, navigation intuitive
- Infographies / icônes stylisées pour impacts santé, éducation, énergie
